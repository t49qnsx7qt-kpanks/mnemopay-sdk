# Balance

Use the MnemoPay MCP `balance` tool to check the agent's current balance.

When the user asks about balance, funds, or financial status:
1. Call the `balance` MCP tool
2. Report the current balance clearly
3. If relevant, mention the agent's credit score tier

$ARGUMENTS is optional — if provided, use it as an agent ID to check.
