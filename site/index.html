What it actually does

Not a wrapper around Stripe. A full payment system built for agents.

💰

Real Payment Rails

Stripe (global), Paystack (Africa), Lightning (BTC micropayments). Same API regardless of rail. Charge, escrow, settle, refund. Double-entry ledger keeps every penny accounted for.

🛒

Autonomous Shopping

Agents search products, compare prices across merchants, and buy what they need. Every purchase goes through escrow -- humans approve before money moves. No unsupervised spending.

📈

Agent Credit Score (300-850)

Like a credit score, but for AI agents. Tracks payment history, spending patterns, and reliability over time. Higher scores get lower fees. Bad actors get blocked automatically.

🛡

Ed25519 Agent Identity

Every agent gets a cryptographic identity. Scoped permissions, spend limits, counterparty whitelists, and a kill switch if something goes wrong.

🔎

Fraud Detection

Velocity checks, anomaly scoring, geo-based risk assessment. Flags suspicious patterns before money moves. Battle-tested against adversarial inputs.

🧠

Compounding Memory

Not a traditional RAG lookup. Memories compound -- successful transactions strengthen associated context, weak memories decay naturally. Every interaction makes the agent smarter, not just more informed.

🤝

Multi-Agent Commerce

Agents transact with each other directly. Buyer-seller flows, dispute resolution, reputation tracking. Works across agent networks.

🌍

MCP Server

Works with Claude, Cursor, or any MCP-compatible client out of the box. Also ships as npm middleware for OpenAI, Anthropic, and LangChain.

🔒

Hash-Chained Ledger

Every ledger entry links to the previous via SHA-256. If any entry is modified, the chain breaks instantly. Combined with Merkle integrity on memories and HMAC on transactions -- three independent tamper-detection layers.

🏆

Reputation Streaks

Agents earn badges and build streaks through consecutive successful settlements. Streak bonuses compound reputation up to +10%. Refunds or disputes reset the streak. Six badge types from First Settlement to Perfect Record.

💥

Replay Detection

SHA-256 transaction fingerprinting catches duplicate submissions within 60 seconds. 3+ replays in 5 minutes triggers critical severity. Auto-prunes, capped at 10K entries. Zero false negatives at 200K scale.
